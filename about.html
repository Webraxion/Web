О платформе

Этот проект демонстрирует мультистраничную веб-платформу с регистрацией, входом, личным кабинетом, форумом, новостной лентой и чатом.

Возможности

  • Регистрация и авторизация пользователей
  • Страница профиля, выход
  • Публикация тем на форуме
  • Публикация новостей
  • Личные сообщения через чат
  • Node.js backend с хранением данных в JSON

Технологии

Проект использует:

  • Node.js + Express
  • Vanilla JavaScript ES modules
  • HTML/CSS
  • JWT + bcrypt
  • Local storage для режима offline
